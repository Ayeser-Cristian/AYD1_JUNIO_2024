### Librerías usadas para este backend

```
npm install express
npm install cors
npm install mysql2
npm install nodemon
npm install dotenv

```
### Levantar el proyecto

```
npm start
```


### Librerías usadas para las pruebas unitarias

```
npm install --save-dev jest

npm install --save-dev supertest
```

### Ejecutar las pruebas unitarias

```
npx jest
```
